# Arkanolf

A futuristic, neon-infused breakout game built entirely with vanilla JavaScript and HTML5 Canvas — no frameworks, no dependencies, no tracking.

**[▶ Play Now](https://xbrasil.github.io/Arkanolf/)**

---

## Features

- **35 levels** with unique hand-crafted layouts and animated backgrounds
- **4 difficulty modes** — Relax, Normal, Hard, and Insane
- **16 power-ups** — laser cannon, multi-ball, fireball, plasma ball, sticky paddle, ice ball, bomb ball, and more
- **Special bricks** — stone bricks that require multiple hits, indestructible bricks, and bomb bricks
- **Mobile-ready** — touch controls and haptic feedback
- **Purely local** — runs entirely in your browser; no server, no login, no data collection

## Power-ups

| Power-up | Effect |
|---|---|
| 🔵 Multi-ball | Splits the ball into three |
| 🔴 Fireball | Burns through bricks |
| 🟣 Plasma Ball | Passes through multiple bricks |
| 🧊 Ice Ball | Freezes bricks on impact |
| 💣 Bomb Ball | Explodes nearby bricks |
| 😈 Wicked Ball | Unpredictable bouncing |
| 🪨 Stone Ball | Breaks indestructible bricks |
| ⚡ Laser | Paddle fires laser beams |
| 🏏 Big Paddle | Widens the paddle |
| 🫧 Sticky Paddle | Ball sticks on contact |
| 💛 Extra Life | +1 life |
| ⬆️ Fast Ball | Increases ball speed |
| ⬇️ Slow Ball | Decreases ball speed |
| 🎱 Huge Ball | Massive ball |
| 🔵 Tiny Ball | Shrinks the ball |
| 💀 Cursed Skull | Something bad happens... |

## How to Play

- **Desktop** — move the paddle with the mouse; click to launch the ball
- **Mobile** — drag with your finger; tap to launch
- Break all breakable bricks to complete each level
- Collect falling power-ups to gain (or lose) abilities
- Progress through difficulty tiers to unlock higher levels

## Stack

Pure HTML5 + Canvas + Vanilla JavaScript. No build step, no bundler, no runtime dependencies.

```
index.html          ← game entry point
static/
  js/               ← game modules (physics, graphics, levels, menus…)
  css/styles.css    ← styling
  images/           ← textures and icons
```
